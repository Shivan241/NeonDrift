# 🎮 About

NEONDRIFT is a modern cyberpunk-inspired endless highway racing game that runs entirely inside the browser.

Drive through procedurally generated highways, avoid traffic, collect coins, unlock new vehicles, upgrade performance, and chase your highest score in an endless racing experience.

No installation required.

Just open the game and race.

# 🕹 Controls

## Desktop

| Key | Action |
|------|---------|
| ← → / A D | Steering |
| Shift | Nitro |
| C | Change Camera |
| Space | Nitro |

---

## Mobile

- Left Touch → Move Left
- Right Touch → Move Right
- Nitro Button → Boost
